# FastApi DataStorage

REST API сервис, для хранения метадаты и данных CSV, XLSX файлов и их выгрузки.

- На backend используется: **FastApi** 
- В качестве БД : **PostgreSQL**
- Для запросов к БД и создание моделей используется: **SQLAlchemy**
